Setup — Power Automate Connection

Configure the connection to your SharePoint Lists via Power Automate. Do this once — settings are saved in your browser.

Step 1 — Paste your Company Flow URL

After creating the Power Automate flow for Companies, copy the HTTP POST URL from the trigger step and paste it here.

Step 2 — Paste your Contact Flow URL

After creating the Power Automate flow for Contacts, copy the HTTP POST URL from the trigger step and paste it here.

How to get the Flow URL:
In Power Automate → open your flow → click the When a HTTP request is received trigger → copy the HTTP POST URL that appears. It will only appear after you save the flow for the first time. See the setup guide document for the full step-by-step.
Offline / backup mode: If the Flow URL is not configured, all form submissions are saved locally in your browser and you can export them as a CSV using the Export button on each tab. This CSV can be imported directly into SharePoint or HubSpot.
